Croxx - A Cross Platform Game Engine

Croxx is an upcoming Cross Platform Game Engine, written in C++ (with minimal platform specific native coding) and using OpenGL ES and Direct3D. It is intended to support Windows, Windows Mobile, Android and iOS.
